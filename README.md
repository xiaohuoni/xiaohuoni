# 关于我

## xiaohuoni 如何断词？

xiaohuoni = xiaohu + oni 小虎是我初中开始用的笔名，oni 是一个拟声词，替代一种不常见的少数名族方言，是“心中的太阳”的意思。是我的乳名。
